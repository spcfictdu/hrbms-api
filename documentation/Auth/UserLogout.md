### User Logout

This endpoint is used to logout the user.

#### URL

```
{{base_url}}/api/user/logout
```   

#### Method
```
GET
```

#### Authentication Needed
```
TRUE
```

#### Permitted Roles
```
NO ROLE NEEDED
```

#### Response Example (Success)

``` json
{
    "message": "Logged out successfully",
    "results": [],
    "code": 200,
    "error": false
}
```
