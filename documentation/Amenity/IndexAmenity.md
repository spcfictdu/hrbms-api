### Index Amenity

This endpoint retrieves the list of all amenities.

#### URL

```
{{base_url}}/api/amenity/
```

#### Method

```
GET
```

#### Authentication Needed

```
TRUE
```

#### Permitted Roles

```
ADMIN
```
